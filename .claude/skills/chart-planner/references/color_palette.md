# Color Palette

## ノード種別別デフォルトカラー

| kind | fill | stroke | stroke_width | 用途 |
|---|---|---|---|---|
| start | `#BFE9D6` (薄緑) | `#1a1a1a` | 2.0 | 開始ノード |
| end | `#DDDDDD` (グレー) | `#1a1a1a` | 2.0 | 終了ノード |
| task | `#FFFFFF` (白) | `#1a1a1a` | 2.0 | 通常タスク |
| decision | `#FFF3CD` (薄黄) | `#1a1a1a` | 2.0 | 判断ポイント |
| chip | `#D9ECFF` (薄青) | `#7AA7D9` | 1.5 | システムラベル |

## 強調カラー（特別な意味を持つノード）

| 意味 | fill | 用途例 |
|---|---|---|
| エラー/修正 | `#F9D7D7` (薄赤) | 差異修正、エラー処理タスク |
| 承認/完了 | `#BFE9D6` (薄緑) | 承認済み、完了タスク |
| 注意/確認 | `#FFF3CD` (薄黄) | 確認が必要なタスク |
| 情報/参照 | `#D9ECFF` (薄青) | 参照のみのタスク |

## エッジカラー

| 意味 | color | style | 用途 |
|---|---|---|---|
| 通常フロー | `#1a1a1a` (黒) | solid | 順方向の通常接続 |
| Yes / 肯定 | `#2E7D32` (緑) | solid | 判断のYes分岐 |
| No / 否定 | `#C62828` (赤) | solid | 判断のNo分岐 |
| 差戻し/ループ | `#C62828` (赤) | dashed | 逆流エッジ |
